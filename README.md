# COMP1008 - Assignment 2
### A Student Course Management System

This project focuses on implementing a management system for student courses. When using the system, users will be able to add students, assign courses, validate input, display student details, and process/manage data.

When the program is ran, the user can choose from the following options:
1. Add a new student
2.  Remove a student by ID
3. Display all students
4. Search a student by ID or name
5. Add a course to a student's courses
6. Drop a student's course
7. Quit

### Adding a new student
Adding a new student requires a valid name, student ID (S-XXXX), and an email. This will add the student into the system.

### Removing a student by ID
To remove a student, the user must provide a valid student ID (S-XXXX). If the student exists in the system, they will be removed.

### Displaying all students
This option will display a list of all students currently in the system, along with their details such as name, student ID, email, and courses.

### Searching a student by ID or name
Users can search for a student by providing either their student ID or name. The system will return the details of the student if found, or an appropriate message if the student does not exist.

### Adding a course to a student's courses
To add a course to a student's list of courses, the user must provide the student's ID or name and the course code. The system will validate the student ID and add the course to the student's record if the student exists.

### Dropping a student's course
To drop a course from a student's list of courses, the user must provide the student's ID or name and the course code. The system will validate the student ID and remove the course from the student's record if the student exists and is enrolled in the course.

### Quitting the program
When the user chooses to quit the program, simply enter [7] to exit the system.
